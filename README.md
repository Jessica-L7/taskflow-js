# TaskFlow JS

Responsive task management web application built with HTML5, CSS3 and vanilla JavaScript.

TaskFlow JS allows users to add, complete and delete tasks with persistent browser storage using LocalStorage. The project was built to strengthen frontend development skills including DOM manipulation, event handling, responsive UI design and dynamic state updates.

## Features

- Add and remove tasks
- Mark tasks as completed
- Persistent task storage using LocalStorage
- Dynamic active task counter
- Keyboard support using Enter key submission
- Responsive minimalist UI
- Real-time DOM updates
- Input validation using `.trim()`
- Dynamic singular/plural task grammar logic

## Development Iterations

During development, several frontend improvements and UI refinements were implemented:

- Added dynamic active task counter functionality
- Refined task counter positioning using CSS absolute positioning
- Improved task counter visibility through iterative UI styling updates
- Added keyboard-friendly task submission using Enter key event handling
- Added conditional singular/plural grammar logic (`1 task` vs `2 tasks`)
- Updated counter logic to dynamically exclude completed tasks
- Improved form validation using `.trim()` to prevent empty space submissions
- Refined layout spacing and visual hierarchy for improved user experience
